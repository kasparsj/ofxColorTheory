Supported color schemes:

* Analogous
* Complementary
* SplitComplementary
* Compound
* FlippedCompound
* Monochrome
* Tetrad
* Triad

Interpolation is done using [CIELCh](https://en.wikipedia.org/wiki/CIELAB_color_space#Cylindrical_representation:_CIELCh_or_CIEHLC) color space.
